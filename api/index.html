    Function reference

    Aggregations

    agg.any_value Returns the first non-null value from arg.
    agg.approx_count_distinct Gives the approximate count of distintinct elements using HyperLogLog.
    agg.approx_quantile Gives the approximate quantile using T-Digest.
    agg.arg_max Finds the row with the maximum val. Calculates the arg expression at that row.
    agg.arg_min Finds the row with the minimum val. Calculates the arg expression at that row.
    agg.avg Calculates the average value for all tuples in arg.
    agg.bit_and Returns the bitwise AND of all bits in a given expression .
    agg.bit_or Returns the bitwise OR of all bits in a given expression.
    agg.bit_xor Returns the bitwise XOR of all bits in a given expression.
    agg.bool_and Returns TRUE if every input value is TRUE, otherwise FALSE.
    agg.bool_or Returns TRUE if any input value is TRUE, otherwise FALSE.
    agg.corr Returns the correlation coefficient for non-null pairs in a group.
    agg.count Calculates the number of tuples tuples in arg.
    agg.covar_pop Returns the population covariance of input values.
    agg.entropy Returns the log-2 entropy of count input-values.
    agg.favg Calculates the average using a more accurate floating point summation (Kahan Sum).
    agg.first Returns the first value of a column.
    agg.fsum Calculates the sum using a more accurate floating point summation (Kahan Sum).
    agg.histogram Returns a LIST of STRUCTs with the fields bucket and count.
    agg.kurtosis Returns the excess kurtosis of all input values.
    agg.last Returns the last value of a column.
    agg.list Returns a LIST containing all the values of a column.
    agg.mad Returns the median absolute deviation for the values within x. NULL values are ignored. Temporal types return a positive INTERVAL.
    agg.max Returns the maximum value present in arg.
    agg.median Returns the middle value of the set. NULL values are ignored. For even value counts, quantitiative values are averaged and ordinal values return the lower value.
    agg.min Returns the minumum value present in arg.
    agg.mode Returns the most frequent value for the values within x. NULL values are ignored.
    agg.product Calculates the product of all tuples in arg
    agg.quantile_cont Returns the intepolated quantile number between 0 and 1 . If pos is a LIST of FLOATs, then the result is a LIST of the corresponding intepolated quantiles.
    agg.quantile_disc Returns the exact quantile number between 0 and 1 . If pos is a LIST of FLOATs, then the result is a LIST of the corresponding exact quantiles.
    agg.regr_avgx Returns the average of the independent variable for non-null pairs in a group, where x is the independent variable and y is the dependent variable.
    agg.regr_avgy Returns the average of the dependent variable for non-null pairs in a group, where x is the independent variable and y is the dependent variable.
    agg.regr_count Returns the number of non-null number pairs in a group.
    agg.regr_intercept Returns the intercept of the univariate linear regression line for non-null pairs in a group.
    agg.regr_r2 Returns the coefficient of determination for non-null pairs in a group.
    agg.regr_slope Returns the slope of the linear regression line for non-null pairs in a group.
    agg.regr_sxx -
    agg.regr_sxy Returns the population covariance of input values.
    agg.regr_syy -
    agg.reservoir_quantile Gives the approximate quantile using reservoir sampling, the sample size is optional and uses 8192 as a default size.
    agg.skewness Returns the skewness of all input values.
    agg.stddev_pop Returns the population standard deviation.
    agg.stddev_samp Returns the sample standard deviation.
    agg.string_agg Concatenates the column string values with a separator
    agg.sum Calculates the sum value for all tuples in arg.
    agg.var_pop Returns the population variance.
    agg.var_samp Returns the sample variance of all input values.

    Blobs

    blob.decode Convert blob to varchar. Fails if blob is not valid utf-8.
    blob.encode Convert varchar to blob. Converts utf-8 characters into literal encoding.
    blob.octet_length Number of bytes in blob

    Datetime

    dt.age Subtract arguments, resulting in the time difference between the two timestamps
    dt.century Extracts the century of a timestamp
    dt.current_date Current date (at start of current transaction)
    dt.current_localtimestamp Returns a TIMESTAMP whose GMT bin values correspond to local date and time in the current time zone.
    dt.date_diff The number of partition boundaries between the dates
    dt.date_part Get the subfield (equivalent to extract)
    dt.date_sub The number of complete partitions between the dates
    dt.date_trunc Truncate to specified precision
    dt.datediff Alias of date_diff. The number of partition boundaries between the dates
    dt.datepart Alias of date_part. Get the subfield (equivalent to extract)
    dt.datesub Alias of date_sub. The number of complete partitions between the dates
    dt.datetrunc Alias of date_trunc. Truncate to specified precision
    dt.day Day
    dt.dayname The (English) name of the weekday
    dt.dayofmonth Day (synonym)
    dt.dayofweek Numeric weekday (Sunday = 0, Saturday = 6)
    dt.dayofyear Numeric ISO weekday (Monday = 1, Sunday = 7)
    dt.decade Decade (year / 10)
    dt.epoch Converts a timestamp to the seconds since the epoch
    dt.epoch_ms Converts ms since epoch to a timestamp
    dt.era Calendar era
    dt.generate_series Generate a table of timestamps in the closed range, stepping by the interval
    dt.get_current_time Current time (start of current transaction)
    dt.get_current_timestamp Current date and time (start of current transaction)
    dt.hour Hours
    dt.isodow Numeric ISO weekday (Monday = 1, Sunday = 7)
    dt.isoyear ISO Year number (Starts on Monday of week containing Jan 4th)
    dt.last_day The last day of the corresponding month in the date
    dt.make_date The date for the given parts
    dt.make_time The time for the given parts
    dt.make_timestamp The timestamp for the given parts
    dt.make_timestamptz The timestamp with time zone for the given parts in the current time zone
    dt.microsecond Sub-minute microseconds
    dt.millennium Millennium
    dt.millisecond Sub-minute milliseconds
    dt.minute Minutes
    dt.month Month
    dt.monthname The (English) name of the month
    dt.now Current date and time (start of current transaction)
    dt.quarter Quarter
    dt.range Generate a table of timestamps in the half open range, stepping by the interval
    dt.second Seconds
    dt.strftime Converts a date to a string according to the format string
    dt.strptime Converts string to timestamp according to the format string
    dt.time_bucket Truncate date by the specified interval bucket_width. Buckets are aligned relative to origin date. origin defaults to 2000-01-03 for buckets that don’t include a month or year interval, and to 2000-01-01 for month and year buckets.
    dt.timezone Use the date parts of the timestamp in GMT to construct a timestamp in the given time zone. Effectively, the argument is a “local” time.
    dt.timezone_hour Time zone offset hour portion
    dt.timezone_minute Time zone offset minutes portion
    dt.to_days Construct a day interval
    dt.to_hours Construct a hour interval
    dt.to_microseconds Construct a microsecond interval
    dt.to_milliseconds Construct a millisecond interval
    dt.to_minutes Construct a minute interval
    dt.to_months Construct a month interval
    dt.to_seconds Construct a second interval
    dt.to_timestamp Converts sec since epoch to a timestamp
    dt.to_years Construct a year interval
    dt.today Current date (start of current transaction)
    dt.transaction_timestamp Current date and time (start of current transaction)
    dt.week ISO Week
    dt.weekday Numeric weekday synonym (Sunday = 0, Saturday = 6)
    dt.weekofyear ISO Week (synonym)
    dt.year Year
    dt.yearweek BIGINT of combined ISO Year number and 2-digit version of ISO Week number

    JSON

    json.array_to_json Alias for to_json that only accepts LIST
    json.from_json Alias for json_transform
    json.from_json_strict Alias for json_transform_strict
    json.json_array Create a JSON array from any number of values
    json.json_array_length Return the number of elements in the JSON array json, or 0 if it is not a JSON array. If path is specified, return the number of elements in the JSON array at the given path. If path is a LIST, the result will be LIST of array lengths
    json.json_contains Returns true if json_needle is contained in json_haystack. Both parameters are of JSON type, but json_needle can also be a numeric value or a string, however the string must be wrapped in double quotes
    json.json_extract Extract JSON from json at the given path. If path is a LIST, the result will be a LIST of JSON
    json.json_extract_string Extract VARCHAR from json at the given path. If path is a LIST, the result will be a LIST of VARCHAR
    json.json_keys Returns the keys of json as a LIST of VARCHAR, if json is a JSON object. If path is specified, return the keys of the JSON object at the given path. If path is a LIST, the result will be LIST of LIST of VARCHAR
    json.json_merge_patch Merge two json documents together
    json.json_object Create a JSON object from any number of key, value pairs
    json.json_quote Alias for to_json
    json.json_structure Return the structure of json. Throws an error if the structure is inconsistent (incompatible types in an array)
    json.json_transform Transform json according to the specified structure
    json.json_transform_strict Same as json_transform, but throws an error when type casting fails
    json.json_type Return the type of the supplied json, which is one of OBJECT, ARRAY, BIGINT, UBIGINT, VARCHAR, BOOLEAN, NULL. If path is specified, return the type of the element at the given path. If path is a LIST, the result will be LIST of types
    json.json_valid Return whether json is valid JSON
    json.row_to_json Alias for to_json that only accepts STRUCT
    json.to_json Create JSON from a value of any type. Our LIST is converted to a JSON array, and our STRUCT and MAP are converted to a JSON object

    nested.list_sort Sorts the elements of the list. See the Sorting Lists section for more details about the sorting order and the null sorting order.

    nested.list_unique Counts the unique elements of a list.
    nested.list_value Create a LIST containing the argument values.
    nested.map Returns an empty map.
    nested.map_extract Alias of element_at. Return a list containing the value for a given key or an empty list if the key is not contained in the map. The type of the key provided in the second parameter must match the type of the map’s keys else an error is returned.
    nested.map_from_entries Returns a map created from the entries of the array
    nested.row Create a STRUCT containing the argument values. If the values are column references, the entry name will be the column name; otherwise it will be the string ‘vN’ where N is the (1-based) position of the argument.
    nested.struct_extract Extract the named entry from the struct.
    nested.struct_insert Add field(s)/value(s) to an existing STRUCT with the argument values. The entry name(s) will be the bound variable name(s).
    nested.struct_pack Create a STRUCT containing the argument values. The entry name will be the bound variable name.
    nested.union_extract Extract the value with the named tags from the union. NULL if the tag is not currently selected
    nested.union_tag Retrieve the currently selected tag of the union as an Enum.
    nested.union_value Create a single member UNION containing the argument value. The tag of the value will be the bound variable name.

    num.acos computes the arccosine of x
    num.asin computes the arcsine of x
    num.atan computes the arctangent of x
    num.atan2 computes the arctangent (x, y)
    num.bit_count returns the number of bits that are set
    num.cbrt returns the cube root of the number
    num.ceil rounds the number up
    num.ceiling rounds the number up. Alias of ceil.

    num.cos computes the cosine of x
    num.cot computes the cotangent of x
    num.degrees converts radians to degrees
    num.even round to next even number by rounding away from zero.
    num.factorial See ! operator. Computes the product of the current integer and all integers below it
    num.floor rounds the number down
    num.gamma interpolation of (x-1) factorial (so decimal inputs are allowed)
    num.greatest selects the largest value
    num.isfinite Returns true if the floating point value is finite, false otherwise
    num.isinf Returns true if the floating point value is infinite, false otherwise
    num.isnan Returns true if the floating point value is not a number, false otherwise
    num.least selects the smallest value
    num.lgamma computes the log of the gamma function.
    num.ln computes the natural logarithm of x
    num.log computes the 10-log of x
    num.log10 alias of log. computes the 10-log of x
    num.log2 computes the 2-log of x
    num.nextafter return the next floating point value after x in the direction of y
    num.pi returns the value of pi
    num.pow computes x to the power of y
    num.power Alias of pow. computes x to the power of y
    num.radians converts degrees to radians
    num.random returns a random number between 0 and 1
    num.round round to s decimal places
    num.setseed sets the seed to be used for the random function
    num.sign returns the sign of x as -1, 0 or 1
    num.signbit returns whether the signbit is set or not
    num.sin computes the sin of x
    num.sqrt returns the square root of the number
    num.tan computes the tangent of x
    num.xor bitwise XOR

    str.md5 Returns the MD5 hash of the value
    str.mismatches The number of positions with different characters for 2 strings of equal length. Different case is considered different.
    str.nfc_normalize Convert string to Unicode NFC normalized string. Useful for comparisons and ordering if text data is mixed between NFC normalized and not.
    str.not_like_escape Returns false if the string matches the like_specifier (see Pattern Matching). escape_character is used to search for wildcard characters in the string.
    str.ord Return ASCII character code of the leftmost character in a string.
    str.position Return location of first occurrence of search_string in string, counting from 1. Returns 0 if no match found.
    str.prefix Return true if string starts with search_string.
    str.printf Formats a string using printf syntax
    str.regexp_extract Split the string along the regex and extract first occurrence of group
    str.regexp_full_match Returns true if the entire string matches the regex (see Pattern Matching)

    str.regexp_split_to_array Alias of string_split_regex. Splits the string along the regex
